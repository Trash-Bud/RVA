# RVA - Project 1

## Building using pip

    python -m venv venv
    source venv/bin/activate  # venv/Scripts/Activate.ps1 if on Windows
    pip install .

Run with `rva-proj1` (if the venv is activated) or `venv/bin/rva-proj1`.

## Building using rye

    rye sync

Run with `rye run rva-proj1`.
